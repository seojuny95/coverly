# 샘플 보험증권

Coverly의 업로드·보험 분석·보장 합계·중복 확인·AI 상담을 로컬에서 확인하기 위한 가상 보험증권이다. 실제 보험상품이나 개인정보를 사용하지 않았으며, 보험 계약이나 보험금 청구 판단에 사용할 수 없다.

## 파일 구성

| 파일                            | 확인할 수 있는 기능                                            |
| ------------------------------- | -------------------------------------------------------------- |
| `01_건강보험_기본형.pdf`        | 암·유사암·뇌·심장·수술·입원·사망 보장과 첫 번째 실손 계약      |
| `02_건강보험_추가형.pdf`        | 정액형 보장 합계와 출처, 두 번째 실손 계약에 따른 중복 신호    |
| `03_자동차_운전자_복합보험.pdf` | 자동차 배상·자차·상해와 운전자 형사합의·변호사·벌금 점검       |
| `04_여행_화재_복합보험.pdf`     | 해외 의료비·휴대품·항공기 지연과 주택 화재·배상·복구 비용 점검 |

네 파일을 모두 업로드하면 월납으로 확인된 보험료 합계는 **109,000원**이다. 연납인 자동차·운전자 복합보험의 720,000원은 월납 합계에서 제외된다.

정액형 담보는 암진단비 5,000만원, 뇌혈관질환진단비 2,000만원, 허혈성심질환진단비 2,000만원, 질병수술비 150만원으로 합산되는 흐름을 확인할 수 있다. 질병입원일당은 금액 합계가 아니라 계약별 1일당 3만원·2만원으로 표시된다. `질병실손의료비`와 `상해실손의료비`는 두 계약에 반복되지만 실제 손해를 넘겨 합산하지 않고 실손 중복 확인 대상으로 분리된다.

## 추천 질문

- 가입한 보험은 총 몇 개고 월 보험료 합계는 얼마야?
- 암 진단비와 유사암 진단비는 각각 얼마나 있어?
- 뇌혈관질환과 심장질환 진단비는 모두 합쳐 얼마야?
- 질병수술비와 입원일당은 계약별로 어떻게 달라?
- 실손의료보험이 중복되어 있어? 두 계약을 모두 유지해야 해?
- 사망 보장은 어떤 담보로 확인돼?
- 다른 차를 긁었을 때와 내 차가 파손됐을 때 각각 어떤 담보를 봐야 해?
- 무보험차 사고로 다치면 어떤 보장이 있어?
- 교통사고 형사합의금, 변호사 비용, 벌금 담보가 있어?
- 해외에서 병원에 가거나 휴대품을 도난당하면 어떤 보장이 있어?
- 비행기가 네 시간 넘게 지연되면 확인할 담보가 있어?
- 집에 불이 나거나 이웃집까지 피해가 생기면 어떤 담보를 봐야 해?
- 첫 번째 건강보험의 암 보장개시일과 감액 조건은 어떻게 돼?
- 실손의료비 자기부담금과 갱신 조건은 어떻게 적혀 있어?
